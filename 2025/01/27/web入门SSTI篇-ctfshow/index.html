  1. **{% set ... %}**：这是 Jinja2 中的语法，用于定义一个变量。这里我们正在创建一个变量 po。

  3. **|join**：join 是一个 Jinja2 过滤器。它用于将可迭代对象（如列表或字典的值）连接成一个字符串，通常使用指定的分隔符（如果没有提供，默认是空字符串）。在这个例子中，join 会尝试将字典的键或值连接成一个字符串。
