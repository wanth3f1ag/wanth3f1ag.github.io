    0x01漏洞分析

    CC4其实就是CommonsCollections4版本的反序列化漏洞的链子，而之前的CC1、CC3、CC6都是用的CommonsCollections <= 3.1.2的版本，我们这里就不过多介绍了

    0x02影响版本&环境搭建

    CC：Commons-Collections 4.0

    jdk版本：jdk8u65

    环境搭建的话，直接在之前的maven项目的pom.xml中添加版本就行
