并且后续加载class的代码都并没有加载成功，来到未开启AutoType的逻辑

先进行一轮黑名单的检测和白名单的匹配，这里我们的类不在黑名单中，所以正常步过

使用ParserConfig自身的类加载器去尝试读取目标类的class文件字节码，这里用的是ASM库去读取字节码，随后尝试获取JsonType注解信息，但是这里并不会获取到，所以是false
